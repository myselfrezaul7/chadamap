# 📍 চাঁদা-ম্যাপ (Chanda-Map)

**Waze-এর মতো চাঁদাবাজ ট্র্যাকার অ্যাপ! বাংলাদেশের নেক্সট বিলিয়ন ডলার স্টার্টআপ?**

A satirical SaaS landing page and admin dashboard for a fictional crowdsourced extortion-tracking startup in Bangladesh. Entirely in **Bengali (Bangla)** with a dark cyberpunk aesthetic.

> ⚠️ **Disclaimer:** চাঁদা-ম্যাপ একটি স্যাটায়ার কনসেপ্ট। দয়া করে নিজ দায়িত্বে চাঁদা প্রদান করুন। 😉

## ✨ Features

- 🗺️ **Hero Section** — 3D isometric Dhaka street map with animated stats
- 🚀 **Premium Features Grid** — চাঁদা estimation, mood analysis, danger zones, VIP mama button
- 📝 **Crowdsourced Report Form** — With mood radios and toast notifications
- 💼 **Enterprise Pricing** — সিন্ডিকেট প্রো প্ল্যান at ৯,৯৯৯ ৳/month
- 🖥️ **Admin Dashboard** — Password-protected with KPI cards, data table, heatmap, God Mode
- 📜 **Satirical Legal Terms** — In Bengali

## 🛠️ Tech Stack

- Vanilla HTML, CSS, JavaScript
- Google Fonts (Orbitron + Inter)
- No frameworks, no dependencies

## 🔒 Admin Dashboard

The admin panel is password-protected. Default password: `password`

To change it, update the `ADMIN_HASH` in `script.js` with a new SHA-256 hash.

## 🚀 Running Locally

```bash
npx http-server . -p 8080
```

Then open `http://localhost:8080`
